# OmniConvert

OmniConvert is a powerful, web-based unit conversion tool designed for accuracy, speed, and ease of use. It handles a wide range of measurement categories, from basic physical dimensions to complex data storage and real-time currency rates.

## Features

- **Extensive Unit Support**: Convert between units in categories including:
  - Length
  - Weight
  - Temperature
  - Area
  - Volume
  - Speed
  - Time
  - Data Storage
  - Energy
  - Pressure
  - Force
  - Currency (Live exchange rates)
- **Custom Units**: Define and save your own custom conversion units.
- **Conversion History**: Easily track your recent conversions.
- **Visual Analytics**: View conversion trends with interactive line charts.
- **Adjustable Precision**: Configure the decimal precision for results (0-5 places).
- **Scientific Notation**: Automatically handles very large or very small numbers using scientific notation.
- **Modern UI**: Clean, high-contrast dark mode design for optimal readability.

## Technologies Used

- **Frontend**: React 18+ with Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Visualization**: Recharts
- **Backend**: Node.js/Express (for API proxying)

## Setup and Running

1. **Prerequisites**: Ensure you have [Node.js](https://nodejs.org/) installed.
2. **Install Dependencies**:
   ```bash
   npm install
   ```
3. **Running in Development**:
   ```bash
   npm run dev
   ```
4. **Building for Production**:
   ```bash
   npm run build
   ```
